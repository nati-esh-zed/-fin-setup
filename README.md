# @nez-fin/setup

Project setup script for @nez-fin/core

## Getting Started

```sh
npm install @nez-fin/setup -g
```

```sh
mkdir your-project-name
cd your-project-name
npm init -y
```

```sh
setup-fin 
```

or

```sh
setup-fin --typescript-template
```

This will create a project with a webpack template.

***Happy coding!***
